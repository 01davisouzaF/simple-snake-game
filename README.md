# Snake Game

This repository contains a snake game made using HTML, CSS and JavaScript. We can control the snake with the arrows keys.

## Technologies
- HTML.
- CSS.
- JavaScript.
- Canvas API.
  
## How controller the snake
↑: Move snake up.
↓: Move snake down.
→: Move snake right.
←: Move snake left.
